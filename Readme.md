# React Redux Demo

Project to demonstrate redux with React and TypeScript

## Run Locally

Clone the project

```bash
  git clone https://github.com/andresdavidsv/react-redux-demo
```

Go to the project directory

```bash
  cd react-redux-demo
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
